# AdScoreAI — Build Pack

This folder contains the documents needed to start building AdScoreAI in Cursor.

## Files
- `FINAL_PRD.md` — final product requirements document
- `CURSOR_BUILD_PLAN.md` — practical MVP build order
- `SYSTEM_DESIGN.md` — implementation-oriented architecture notes
- `SCHEMA_BLUEPRINT.md` — suggested data model
- `ROUTES_AND_PAGES.md` — app routes and UI structure
- `CURSOR_MASTER_PROMPT.md` — prompt to paste into Cursor for code generation
- `.env.example` — required environment variables

## Recommended first step
Open `CURSOR_MASTER_PROMPT.md` in Cursor and use it to generate the initial app scaffold.
